# Life OS Obsidian 模板

这是一个已经预装 Life OS 插件的 Obsidian vault 模板。

如果你是第一次使用，建议先打开根目录的 `用户手册.md`。那是一份完整说明，从配置 AI 到日记、任务、记忆、报告和考公模块都有详细步骤。

## 使用方法

1. 解压这个文件夹。
2. 打开 Obsidian。
3. 选择“打开本地仓库”，并选择解压后的文件夹。
4. 如果 Obsidian 提示是否信任此 vault，请确认信任后启用社区插件。
5. 点击左侧工具栏的 Life OS 图标，或在命令面板中搜索 `Life OS` / `Personal Life System`。

## AI 配置

模板不包含任何 API Key。需要使用 AI 日记、Chat、总结、记忆提取或日报能力时，请进入：

`设置 -> 社区插件 -> 个人人生系统 / Life OS`

然后填写你的模型服务、Base URL、模型名和 API Key。

## 目录

- `PersonalLifeSystem/Daily`：日记
- `PersonalLifeSystem/Inbox`：临时收集
- `PersonalLifeSystem/Tasks`：待办
- `PersonalLifeSystem/Memory`：长期记忆
- `PersonalLifeSystem/Reviews`：复盘
- `PersonalLifeSystem/Exam`：考公备考
- `PersonalLifeSystem/Chat`：聊天记录
- `PersonalLifeSystem/Reports`：日报、周报、月报
- `PersonalLifeSystem/Templates`：模板

所有业务数据都以 Markdown 保存，你可以直接编辑、备份或迁移。
